变量名

和常见的语言一样，由数字字母下划线组成，不能以数字开头，不能使用关键字，大小写严格。

PEP 8要求：

  • 用小写字母拼写，多个单词用下划线连接。
  • 受保护的实例属性用单个下划线开头（后面会讲到）。
  • 私有的实例属性用两个下划线开头（后面会讲到）。

变量的使用

可以使用type()函数检查变量的类型，js里是typeof()。

可以使用Python中内置的函数对变量类型进行转换。这里也和js有点像都是使用自己类型的函数进行转换。

  • int()：将一个数值或字符串转换成整数，可以指定进制。
  • float()：将一个字符串转换成浮点数。
  • str()：将指定的对象转换成字符串形式，可以指定编码。
  • chr()：将整数转换成该编码对应的字符串（一个字符）。
  • ord()：将字符串（一个字符）转换成对应的编码（整数）。

函数input(‘a = ‘) 接受控制台输入的内容，可以在函数内自定义内容作为控制台的提示。在使用的时候使用的占位符%d %f等，这个c语言一样。

分支结构

也就是if - else 进行条件判断来决定执行那一段代码，python里if else后面跟的是冒号:。可以看到下方的代码是没有使用花括号的,python使用不同的缩进的方式来表示不同的代码层次。也即如果需要在if后面执行多条语句只需要保证if判断后的语句的缩进相同即可。

else if的方式 elif 与 else: if[表达式]: 推荐的方式为elif。
