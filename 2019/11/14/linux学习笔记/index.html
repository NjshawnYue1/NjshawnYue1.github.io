linux学习笔记

Posted on 2019-11-14 Edited on 2019-11-20

linux系统映像里以前在南京读书的时候好像没有说过，来了这里半学期的课程，老师上课还是一副不耐烦的样子，挺尴尬的，现在好多技术都是服务器上的，只好再次学起这个系统。这个笔记是看尚硅谷韩顺平老师的视频写的。

我用的是阿里的服务器，内核为centos。

linux基础

Linux的历史和安装过程这里不再多做介绍了,感兴趣的同学可以自己去了解一下。同时它与Windows操作系统的比较也不在多说，视频里介绍的都是表面的不同，对于学过os的同学来说个人觉得没什么意义，让我觉得有意思的是这个吉祥物是个企鹅，如果你有注意到的话可以看见在阿里的控制台上会显示一个小企鹅。下面学习其他内容。

目录结构

层级式的树状结构只有一个根目录，对于Linux来说一切皆为文件,系统中的硬件也会被映射成文件进行管理。

可以看到根目录的文件夹及文件。

/bin存放的内容为常用的命令 /home存放普通用户，每当你在系统中创建一个用户时，目录下就会对应生成这个用户的文件夹。 /root为用户权限的主目录。 /etc配置文件 /usr通常存放应用程序和相关文件。/dev 设备映射文件夹 /media识别u盘光盘等 /mnt通常为共享文件夹。/usr/local 安装软件的目录。

实际操作

我使用的连接终端的工具是Termius，感觉还不错，有命令行也可以上传文件，比较方便。

vi vim

直接搜索vim并安装，在这里上课的时候花过一段时间介绍过，对他的一些基本操作我还有点印象。当我们用vim打开一个文档的时候可以对文档进行浏览，或者使用一些快捷键对文档进行操作。按i进入插入模式，编辑完成后按esc进入命令行模式输入:wq保存并退出 :q直接退出 :q!强制不保存退出。

常用的快捷键：yy 复制当前行 nyy复制光标小的n行 dd删除当前行 ndd删除光标下的n行 / 查找指定的字符串 :set nu/nonu显示取消行号 G跳到尾行 gg跳到首行 撤销操作u n+shift+g跳到第n行

开机，重启注销命令

shutdown

shutdown -h now 立即关机

shutdown -h 1 一分钟后关机

shutdown -r now 立即重启

halt

关机

reboot

重启

sync

将内存中的数据同步，关机之前执行。

用户登录和注销

尽量不使用root登录，而要使用普通用户再使用命令su切换权限，logout注销当前用户，命令行下生效。

用户管理

众所周知，linux为多用户操作系统，对权限的管理十分重要，系统中可以有多个用户，不同的用户归于不同的组，不同的组拥有不同的权限，当用户属于那个组的时候也就拥有了对应的权限，所有的用户都在home目录下。

useradd 用户名创建用户后，同时也创建了一个同名的组，这个用户在这个组里，同时/home目录下多了一个文件夹。useradd -d /home/

用户名在指定文件夹下新建一个用户。passwd 用户名 给一个用户指定密码。 userdel 用户名 删除一个用户 userdel -r 用户名删除用户同时删除它的home目录。

id 用户名 查询用户的信息 如果用户不存在就会返回无此用户。

切换用户，权限不够的时候使用su命令进行切换。由高权限切换到低权限不需要输入密码，需要切回原来的用户主需要退出当前的用户即可exit。

用户组

groupadd 组名 / groupdel 组名 useradd -g 用户组 用户名 创建一个在指定组下的用户 usermod -g 用户组 用户名 修改用户的组

用户配置文件

创建的用户在/etc/passwd记录了用户的信息， /etc/group分组 etc/shadow密码

实用指令

这里只记录一些我没怎么用过的命令，常用的cd ls等就不再说了，mkdir -p /a/b 一次可以创建多个目录 不带参数只能创建一级,rmdir删除目录 只能删除空目录 如果目录不为空需要使用命令 rm -rf /a, touch [][][文件1] [文件2] 创建空文件可创建多个，cp 拷贝命令 如果需要拷贝一个目录 则使用参数 cp -r 源文件夹 目标文件夹 如果有相同文件时 在cp前加\ 可以全部覆盖，mv 重命名与移动 在同一个目录和不同目录的区别。cat 查看命令 cat -n 文件 | more 分页显示行号查看 more/less查看文件.

> 与 >>

输出重定向与追加，重定向会覆盖原来的内容而追加不会

ls -l > a.txt 将命令输出的内容重定向到一个文件中,如果文件不存在就会创建。>> 同理

find /home -name/user/size a.txt/a.txt/+/-/20M 不同的查找命令

locate定位，使用之前更新updatedb，因为这个命令基于它自己创建的locate库。

grep命令，通常与其它命令配合使用，比如 cat a.txt | grep -ni a 意思为把查看的结果交给grep查找a这个字符并显示行号忽略大小写，管道操作符| 将前面指令的结果交给后面处理。
