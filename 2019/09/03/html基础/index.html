元素分类

常见的块状元素

常见的内联元素

可以为元素设置属性为

将一个内联元素设置称为一个块状元素。元素独占一行且可以单独设置它的宽高边距等属性，如果不进行设置则会与父容器一致。

而把块元素的属性设置成为inline时就会成为内联元素，内联元素的特点就是元素都在一行并且不可以设置宽高边距等属性。

也可以设置称为inline-block这样一个元素就具有了内联元素和块状元素的特点，即显示在一行并且可以设置宽高间距等属性。

布局

流动模型float

块状元素都会在所处的包含元素内自上而下按顺序垂直延伸分布，因为在默认状态下，块状元素的宽度都为100%**。也就是在不设置宽度的情况下，宽度都占满父元素。实际上，块状元素都会以行的形式占据位置。这种样式是默认的布局模式。内联元素**都会在所处的包含元素内从左到右水平分布显示。

如果需要块状元素在一行内显示只需要为它们设置左右浮动就可以。

层模型 position

绝对定位

如果想为元素设置层模型中的绝对定位,需要设置position:absolute(表示绝对定位)，这条语句的作用将元素从文档流中拖出来，然后使用left、right、top、bottom属性相对于其最接近的一个具有定位属性的父包含块进行绝对定位。如果不存在这样的包含块，则相对于body元素，即相对于浏览器窗口。

相对定位

如果想为元素设置层模型中的相对定位，需要设置position:relative（表示相对定位），它通过left、right、top、bottom属性确定元素在正常文档流中的偏移位置。相对定位完成的过程是首先按static(float)方式生成一个元素(并且元素像层一样浮动了起来)，然后相对于以前的位置移动，移动的方向和幅度由left、right、top、bottom属性确定，偏移前的位置保留不动。

div块本身应该是在左上角的，设置相对定位的时候会依照原本的位置来定位。

固定定位

这个div块会一直在这个位置，滚动条等被拖动时它仍然在这个地方。

Relative与Absolute组合使用

绝对定位在没有设置的情况下是相对Body元素的，当绝对定位于相对定位配合使用的时候才可以让绝对定位基于其他元素进行定位。

父布局设置为相对子布局设置为绝对这样才可以进行想要的定位。

水平居中设置

行内元素进行居中设置的时候只需要设置属性text-align:center即可.:call_me_hand: 但是块状元素就需要分两种情况。

定宽块状：块状元素的width被设置为固定值，满足定宽和块状两个条件的元素是可以通过设置“左右margin”值为“auto”来实现居中的，这两个条件缺一不可。

不定宽块状：在实际开发中更为常见 可以将其设置为行内元素inline或者在外部包装一个table tr td标签。
